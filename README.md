# 开发基础设施Agent化

本仓库致力于构建 **开发基础设施Agent化**，探索如何将 LLM 驱动的智能 Agent 深度融入软件开发的各个环节。我们通过可复用的 Skills（技能插件）将最佳实践沉淀为标准化工具，让 AI Agent 能够像资深工程师一样观察、判断、决策和执行，真正实现开发流程的智能化。
